1.模块的引用

import 模块名

import random
 
random_number = random.randint(1, 99)

import 模块名 as 别名

另外，还可以给模块名取别名

import random as r
 
random_number = r.randint(1, 99)

注意，内置库可以直接引用，如果是第三方库需要先安装，安装方法为：win+r，cmd，pip install 模块名。😎

2.print()函数

print (‘’)让程序无需理解，原样输出引号中的内容。

print()程序需要读懂括号里的内容。

import random
 
random_number = random.randint(1, 99)
print(random_number)
print('random_number')

如果要换行打印怎么办呢？试一试下面的代码。

print('快乐学习编程')
print('每天过得很充实')
print('逐渐走向人生巅峰')
print('''快乐学习编程,
每天过得很充实,
逐渐走向人生巅峰。
''')

3.转义字符

转义字符是用来表示不能直接显示的字符，比如换行键、后退键，回车键等。

print('快乐学习编程,\n每天过得很充实,\n逐渐走向人生巅峰。')

有时候也会遇到一些特殊情况，比如符号重复的问题，如：

print('I'm student in HaiYan')

想一想，为什么程序会报错，应该怎么解决。

print('I\'m student in HaiYan')
print("I'm student in HaiYan")

4.赋值语句

赋值就是把左边的东西装到右边的盒子里。

my_name='小王'
print(name)

想一想，如果依次将两个东西装进同一个’盒子‘里，会怎么样。

my_name='小王'
my_name='小明'
print(name)

被多次赋值时，变量只会储存最新一次的赋值。
Python语言的特色，不需要类型声明，通过赋值语句，就可以定义变量，变量的类型由赋值的结果决定。😎

x = 2
y = 1.0
z = '你好世界'
print(type(x))
print(type(y))
print(type(z))

试一试，55，‘55’，55.0的数据类型分别是什么。

guess_number = input('请输入0到100之间的整数:')

6.字符串拼接

字符串可以通过+号连接起来。

name = input('请输入你的名字：')
print('你好！'+ name)

回到我们的项目中，应该怎么让这个程序更漂亮呢？

randomNumber =  str(random.randint(0,100))
print('系统生成的随机数是：' + randomNumber)

7.数据类型转换

  • str()将其他类型转换为字符串。
  • int()将其他类型转换为整型。
  • float()将其他类型转换为浮点数
    注意：只有内容是整数或浮点数的字符串，才可以被强制转换为整数或浮点数。
    print(int('你好世界'))
    另外，浮点数强制转换为整数时，会直接丢掉小数部分，不是四舍五入。
    print(int(6.8))

    8.条件判断语句

    score = input('请输入你这次Python考了多少分（0-100）:')
 
if score >= 60:
    print('你及格了')
else:  
    print('唉，得加油了')
    思考，上面的程序为什么会报错，应该怎么修改。
    如果有多个条件怎么办呢？

第一种方法：用很多个if（不推荐）

score = int(input('请输入你这次Python考了多少分（0-100）:'))
 
if score >= 90:
    print('不错，很优秀')
if 90 > score >= 60:
	print('还行吧，马马虎虎')
if score < 0:  
    print('唉，得加油了')

第二种方式：（推荐）

score = input('请输入你这次Python考了多少分（0-100）:')
 
if score >= 90:
    print('不错，很优秀')
elif score >= 60:
	print('还行吧，马马虎虎')
else:  
    print('唉，得加油了')

9.while循环

a = 0
while a < 5:
    a = a + 1
    print(a)
while True:
    password = input('请输入密码')
    if password == '8888':
        print('密码正确')
        break
    else:
        print('密码错误')

另一种写法。

password = ''
while password!='8888':
    password=input('请输入密码')
print('密码正确')

回到项目中，用户可以一直猜，直到猜中为止，才结束循环，应该怎么做。

10.异常处理

try:
  #尝试执行这段代码
except:
  #如果出错就执行这段代码，否则不执行

在我们的项目中，如果用户输入的不是数字，会发生什么错误，应该怎么解决呢？

import random
 
random_number = random.randint(1, 99)
print('系统自动产生的随机数：',random_number)
 
while True:
    try:
        guess_number = int(input('请输入0到100之间的数:'))
    except:
        print('你输入的不是数字，请重新输入吧')
    if  guess_number > random_number:
        print('大了')
    elif guess_number < random_number:
        print('小了')
    else:
        print('猜对了')

11.循环控制

  • break 跳出全部循环
  • continue 跳出本次循环
    import random
 
random_number = random.randint(1, 99)
print('系统自动产生的随机数：',random_number)
 
while True:
    try:
        guess_number = int(input('请输入0到100之间的数:'))
    except:
        print('你输入的不是数字，请重新输入吧')
        continue
    if  guess_number > random_number:
        print('大了')
    elif guess_number < random_number:
        print('小了')
    else:
        print('猜对了')
        break

    回到项目中来，如果规定只能猜五次，超过五次就游戏失败，应该怎么做呢？
    import random
 
random_number = random.randint(1, 99)
print('系统自动产生的随机数：',random_number)
 
for i in range(5):
    try:
        guess_number = int(input('请输入0到100之间的数:'))
    except:
        print('你输入的不是数字，请重新输入吧')
        continue
    if  guess_number > random_number:
        print('大了')
    elif guess_number < random_number:
        print('小了')
    else:
        print('猜对了,你一共猜了' + str(i+1) + '次')
        break
else:
    print('猜了5次都没猜到，游戏失败')
