一、列表的使用

1.从列表中提取单个元素

list1 = [0,1,2,3,4]
print(list1[0])

2.从列表中提取多个元素

list1 = [0,1,2,3,4]
print(list1[2:])
print(list1[:2])
print(list1[1:3])
print(list1[2:4])

注意：列表切片遵循原则：“左包右不包”。

3.增加元素

list1 = [0,1,2,3,4]
list1.append(5)
print(list1)

4.删除元素

list1 = [0,1,2,3,4]
list1.remove(1)
print(list1)

5.改变列表中的元素

list1 = [1,2,3,4]
list1[1] = '小明'
print(list1)

6.查找列表中的元素

list1 = ['小明','小红','小孩']
print(list1.index('小红'))

7.in和not in

list = [1,2,3,4,5]
a = 1
 
print(a in list)
print(a not in list)

二、实现剪刀石头布游戏

1.计算机

计算机应该随机选择“剪刀”，“石头”，“布”

import random
 
choice_list = ['石头','剪刀','布']
computer_choice = random.choice(choice_list)
print('计算机出拳：',computer_choice)

2.用户出拳

import random
 
choice_list = ['石头','剪刀','布']
computer_choice = random.choice(choice_list)
print('计算机出拳：',computer_choice)

user_choice = input('请出拳（石头、剪刀、布）：')  # 请用户输入选择
print('用户出拳:',user_choice)
if user_choice not in choice_list:
    print('输入有误，请重新出拳')

判断胜负

import random

#计算机出拳
choice_list = ['石头','剪刀','布']
computer_choice = random.choice(choice_list)
print('计算机出拳：',computer_choice)

#用户出拳
user_choice = ''
while user_choice not in choice_list:
    user_choice = input('请选择（1石头、2剪刀、3布）：')
    if user_choice == '1':
        user_choice = '石头'
    elif user_choice == '2':
        user_choice = '剪刀'
    elif user_choice == '3':
        user_choice = '布'
    else:
        print('输入有误，请重新出拳')
print('用户出拳:',user_choice)

#判断胜负
if user_choice == computer_choice:  # 使用if进行条件判断
	print('平局！')
elif (user_choice == '石头' and computer_choice == '剪刀') or(user_choice == '剪刀' and computer_choice == '布') or (user_choice == '布' and computer_choice == '石头'):
	print('你赢了！')
else:
	print('你输了！')
