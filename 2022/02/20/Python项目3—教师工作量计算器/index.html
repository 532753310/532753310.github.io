一、函数

1.函数的定义与参数传递

先来创建一个最基本的函数吧

def study():
    print('我爱学习编程')
 
study()

我们可以传递一些参数进去

def study(name,school):
    print('我的名字叫做{}，我在{}学习编程'.format(name,school))
 
study('小明','海盐商贸学校')

当然，参数应该按照顺序放进去，不信就试一下下面这段代码。

def study(name,school):
    print('我的名字叫做{}，我在{}学习编程'.format(name,school))
 
study('海盐商贸学校','小明')

我们也可以对参数进行指定，不按照顺序传递。

def study(name,school):
    print('我的名字叫做{}，我在{}学习编程'.format(name,school))
 
study(school='海盐商贸学校',name='小明')

还可以给参数设置默认的值

def study(name,school = '海盐商贸学校'):
    print('我的名字叫做{}，我在{}学习编程'.format(name,school))
 
study(name='小明')

默认参数的值是可以修改的

def study(name,school = '海盐商贸学校'):
    print('我的名字叫做{}，我在{}学习编程'.format(name,school))
 
study('小明','清华大学')

如果不确定参数的数量，我们可以设置一种特殊的不定长参数。

def study(name,school,*course):
    print('我的名字叫做{}，我在{}学习{}'.format(name,school,course))
 
study('小明','海盐商贸学校','编程','数学')

这种不定长参数会返回一种元组类型，如果你觉得不好看，我们可以用循环将其依次取出。

def study(name,school,*course):
    for i in course:
        print('我的名字叫做{}，我在{}学习{}'.format(name,school,i))
 
study('小明','海盐商贸学校','编程','数学')

2.函数的返回值

def wage(month):
    if month < 3:
        money = 5000 * month
    else:
        money = 6000 * month
    return money
 
mouth = int(input('你工作了多少个月:'))
money = wage(month)
print(money)

函数也可以返回多个值，但需要注意，不要依次写多个return，因为遇到第一个return，函数就结束了，后面的程序不会执行。

def wage(month):
    bonus = 1000 * month
    if month < 3:
        money = 5000 * month
    else:
        money = 6000 * month
    return money,bonus

month = int(input('你工作了多少个月:'))
money = wage(month)
print(money)

项目实战

一、任务需求

现海盐商贸学校信息部开展周末技能训练，需要多个老师分工协作完成，竞赛组组长需要一个程序帮他计算工作量，以便安排工作，计算方法有两种。

  • 已知每位老师每月能够完成20课时的工作量。
  • 计算方式1：时间计算
    🐱‍👤例如输入100课时、共有两位老师，程序帮我们计算出需要2.5个月才能完成教学任务。
  • 计算方式2：人力计算
    🐱‍👤例如输入100课时，需要2月之内完成，程序帮我们计算出至少需要3位老师才能完成教学任务。

    二、初始版本

    # 已知每位教师每个月能完成20个课时

#已知总课时，以及现有共有几位教师，计算完成教学任务需要花费多少时间
def JiSuan_month(KeShi,teatherNumber):
    month = KeShi/(teatherNumber*20)
    print("如果一共有%d个课时、并且我们现在有%d个教师，教学任务需要%.1f个月完成，"%(KeShi,teatherNumber,month))

# 已知总课时，规定了完成期限，计算至少需要多少个教师才够
def JiSuan_teatherNumber(KeShi,month):
    KeShi_month = KeShi/month        #每月需要完成多少课时
    teatherNumber = KeShi_month/20   #别忘了每个教师每个月能完成20个课时
    print("如果一共有%d个课时，并且要求在%d个月内完成，我们至少要有%d位教师才够。"%(KeShi,month,teatherNumber))

JiSuan_month(120,2)               #调用时间计算函数
JiSuan_teatherNumber(120,2)       #调用人力计算函数
    注：%f的意思是格式化字符串为浮点型，%.1f的意思是格式化字符串为浮点型，并保留1位小数。

    三、修复BUG

    在使用一段时间之后，我们发现了一个bug，举例来说，假如总课时是30课时，要求一月之内完成，本来应该需要两位老师才够，但是程序计算出来只需要一位老师，原因出在什么地方呢？
    聪明的你应该已经发现了，原来%d在格式化字符串时，自动把小数舍掉了，这种情况我们实际应该需要2位老师，也就是应该向上取整。
    向上取整我们可以使用math.ceil()函数，使用之前记得import。
    # 已知每位教师每个月能完成20个课时
import math
# 已知总课时，以及现有共有几位教师，计算完成教学任务需要花费多少时间
def JiSuan_month(KeShi,teatherNumber):
    month = KeShi/(teatherNumber*20)
    print("如果一共有%d个课时、并且我们现在有%d个教师，教学任务需要%.1f个月完成，"%(KeShi,teatherNumber,month))

# 已知总课时，规定了完成期限，计算至少需要多少个教师才够
def JiSuan_teatherNumber(KeShi,month):
    KeShi_month = KeShi/month        #每月需要完成多少课时
    teatherNumber = math.ceil(KeShi_month/20)   #别忘了每个教师每个月能完成20个课时
    print("如果一共有%d个课时，并且要求在%d个月内完成，我们至少要有%d位教师才够。"%(KeShi,month,teatherNumber))

JiSuan_month(120,2)               #调用时间计算函数
JiSuan_teatherNumber(120,2)       #调用人力计算函数

    四、合并函数

    现在程序已经没有BUG了，但是需要分别调用两个函数，比较麻烦，我们能不能想办法把两个函数合并为一个函数呢？
    # 已知每位教师每个月能完成20个课时
import math

def JiSuan(KeShi,teatherNumber=None,month=None):
    # 如果填了教师的数量，没有填完成期限，则计算所需的时间
    if (teatherNumber != None) and (month == None):
        month = KeShi/(teatherNumber*20)
        print("如果一共有%d个课时、并且我们现在有%d个教师，教学任务需要%.1f个月完成，"%(KeShi,teatherNumber,month))
    # 如果填了完成期限，没有填教师的数量，则计算所需的教师
    elif (teatherNumber == None) and (month != None):
        KeShi_month = KeShi/month                    #每月需要完成多少课时
        teatherNumber = math.ceil(KeShi_month/20)    #别忘了每个教师每个月能完成20个课时
        print("如果一共有%d个课时，并且要求在%d个月内完成，我们至少要有%d位教师才够。"%(KeShi,month,teatherNumber))

JiSuan(KeShi=100,teatherNumber=2)
JiSuan(KeShi=100,month=2)

    六、结构化设计

    现在用户的确可以使用了，但是对于程序员来说，又不符合程序设计的结构性原则，不仅自己很难维护，别人也很难阅读，我们需要做成“子函数+主函数”的代码结构。
    # 已知每位教师每个月能完成20个课时
import math

# 计算
def JiSuan(user_input):
    # 把元祖的数据取出来
    KeShi = user_input[0]
    teatherNumber = user_input[1]
    month = user_input[2]
    # 如果填了教师的数量，没有填完成期限，则计算所需的时间
    if (teatherNumber != None) and (month == None):
        month = KeShi/(teatherNumber*20)
        print("如果一共有%d个课时、并且我们现在有%d个教师，教学任务需要%.1f个月完成，"%(KeShi,teatherNumber,month))
    # 如果填了完成期限，没有填教师的数量，则计算所需的教师
    elif (teatherNumber == None) and (month != None):
        KeShi_month = KeShi/month                     #每月需要完成多少课时
        teatherNumber = math.ceil(KeShi_month/20)    #别忘了每个教师每个月能完成20个课时
        print("如果一共有%d个课时，并且要求在%d个月内完成，我们至少要有%d位教师才够。"%(KeShi,month,teatherNumber))

# 采集用户输入
def userInput():
    user_choice = input('请选择计算的类型(1、教师计算 2、时间计算)：')
    KeShi = int(input('请输入总课时'))
    if user_choice == '1':
        teatherNumber = int(input('请输入现在共有多少位老师(请输入整数):'))
        month = None
    elif user_choice == '2':
        month = float(input('请输入完成期限（月）'))
        teatherNumber = None
    return KeShi,teatherNumber,month
    
# 主函数
def main():
    while True:
        user_input = userInput()
        JiSuan(user_input)

main()

项目实战（食堂情境升级版）

一、任务需求

假设你是联合国主席兼世界首富，想要捐助你的母校海盐商贸学校若干箱鱼子酱，以改善同学们的伙食，但是买得少了不够吃，买得太多吃不完，过了保质期会造成浪费，所以需要一个程序来计算数量，计算方式有两种。

  • 已知每位同学每月能够吃20箱鱼子酱。
  • 计算方式1：时间计算
    🐱‍👤例如输入10000箱鱼子酱、共有200位同学，程序帮我们计算出鱼子酱可以供同学们食用2.5个月。
  • 计算方式2：人力计算
    🐱‍👤例如输入10000箱鱼子酱，但是保质期只有两个月了，程序帮我们计算出至少需要找500位同学才能按时吃完。

def JiSuan_month(yzj_num,student_num):

# 人力计算、已知有食堂买了多少箱鱼子酱，以及鱼子酱的保质期，计算需要找多少位同学来吃，才能在保质期内按时吃完。
def JiSuan_student_num(yzj_num,month):
    student_num = yzj_num/(month*20)

    向上取整我们可以使用math.ceil()函数，使用之前记得import。
