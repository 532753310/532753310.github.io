任务一、密码爆破

目的：

  1. 获取用户信息，为诈骗提供条件
  2. 获得用户网盘里的重要资料（可以提前放点重要文件进去）
  3. 传播反动言论
  4. 传播色情淫秽信息牟利
  5. 实施网络赌博

攻击方法

hydra -l admin -P passw.txt 192.168.148.130 http-post-form "/dvwa/login.php:username=^USER^&password=^PASS^&Login=Login:Login failed" -V

防御：

  1. 密码使用带https的密文传输和存储
  2. 限制错误次数，锁定账户
  3. 增加尝试间隔，sleep(3)
  4. 验证码，图片验证码、短信验证码、二维码验证码
  5. 统计尝试频率，锁定IP地址
  6. token机制
  7. 只能使用二维码登录
  8. 不让用户设置太过于简单的密码
    if(!preg_match('/^[a-zA-Z0-9_*]{6,18}$/',$pw)){
    echo "<script>alert('后端校验：密码只能输入中英文和数字，以及_*，6-18位');history.back()</script>";
    exit();
}

任务二、发包攻击、注册僵尸账号

目的：

  1. 通过僵尸账号非法刷票
  2. 通过网络水军制造舆论，发布虚假信息骗人

防御

  1. 验证码，图片验证码、短信验证码、二维码验证码
  2. 只能通过手机号码注册
  3. 只能通过微信号登录

任务三、XSS跨站脚本攻击

目的：

  1. 盗取用户账号
  2. 控制用户电脑
  3. 通过钓鱼网站实施诈骗

防御：

  1. 不允许用户输入特殊的字符
    // 替换掉<
$gb_word = preg_replace("/</","特殊符号在后端会被我替换掉,再想想别的办法",$gb_word);

任务四、SQL注入

目的：

  1. 获取数据库用户信息，盗取用户账号
  2. 获取考试题目
  3. 篡改考试分数

防御：

  1. 修复漏洞
  2. 对数据库漏洞实施加密
    $sql = "insert into info(username,pw,classNum,className,createtime) 
values ('$username','".md5($pw)."',$classNum,'$className','".time()."')";
$result = mysqli_query($conn,$sql);
if($result){

}

任务五、文件下载漏洞

目的：

  1. 非法获取网站重要资料，如考试题目

防御:

  1. 对文件目录进行加密
  2. 对文件名进行加密

任务六、文件上传漏洞

目的：

  1. 通过上传木马文件，控制服务器，成为肉鸡，进行DOS攻击
  2. 压缩包炸弹，解压最高可达6PB，瞬间拉满服务器负载，进程全炸

任务七、ARP中间人攻击

目的：

  1. 断网攻击（比如使摄像头断网，以便实施作弊）
  2. 获取电脑中重要资料和账号密码

目的：
